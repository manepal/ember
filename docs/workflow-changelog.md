# Workflow Changelog

A log of significant improvements to the AI agent workflows. Helps future agents understand why workflows look the way they do.

---

## 2026-03-03 — Initial workflow creation
- **Why**: Project needed structured guidance for AI agents to contribute effectively
- **What**: Created 5 workflow files: `/develop`, `/new-crate`, `/test`, `/patterns`, `/review`
- **Impact**: Any AI agent can now follow a consistent process for picking up tasks, writing code, testing, and maintaining documentation

## 2026-03-03 — Added self-improvement instructions
- **Why**: Workflows should evolve with the project, not become stale
- **What**: Added "Keep Documentation in Sync" and "Continuously Improve the Agentic Workflow" sections to `/develop`. Added "Update Docs and Workflows" to `/review`.
- **Impact**: Agents now actively improve workflows and docs as they work, compounding quality over time
